Fall Genius Hour

For my fall genius hour I decided to optimize the free to use pcb development website easyeda to make my own pcb. I decided to make a blinking light circuit using a 555 timer integrated circuit. I have a power source that supplies power to the board, three resistors in order to protect the leds from getting too much current. In the end, I really did this project to learn how to use a simple ic with a simple pcb board to learn the basics of electrical egineering.

AP create project

For my ap create project I decided to make a password manager that can store a users passwords but to also remove and find their passwords. The hardest part of this project was being able to figure out how to use the built in json package in python and finding out how to create a json file and store those passwords. The main method I used was converting the json file into a python dictionary, then either adding to that dictionary, deleting something at a specific property name, or finding something as a property name and then converts it back to the json file with the new dictionary.
